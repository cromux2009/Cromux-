Belajar Dasar HTML

HTML adalah bahasa markup yang digunakan untuk membuat web.

Contoh Struktur Dasar HTML:

<!DOCTYPE html>
<html>
    <head>
        <title>Judul Halaman</title>
    </head>
    <body>
        <h1>Halo, dunia!</h1>
    </body>
</html>
